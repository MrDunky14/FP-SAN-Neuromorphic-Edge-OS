# FP-SAN v6 Architectural Whitepaper

This document details the mathematical and biological foundations of the FP-SAN v6 Neuromorphic Operating System, designed for sub-50KB continual learning at the edge.

## 1. The Hybrid 1.58-Bit Matrix (Atomic Physics)
Standard neural networks rely on massive FP32 (32-bit floating-point) matrix multiplications, causing a severe memory bottleneck. FP-SAN v6 compresses synaptic weights into a ternary space: `[-1, 0, 1]`. 

To solve the representation bottleneck (the inability to express continuous gradient distributions in pure ternary), FP-SAN introduces a **Hybrid Scalar**. The heavy array matching is computed via ultra-fast integer addition and subtraction. The final membrane voltage is then multiplied by a single `float cluster_scalar`. This achieves floating-point precision with a 1.58-bit memory footprint.

## 2. Exponential Homeostasis (The Endocrine System)
In un-controlled edge environments, sensor data fluctuates wildly. If ambient energy drops (e.g., a camera lens is obscured or a sensor glitches), standard fixed-threshold Leaky Integrate-and-Fire (LIF) neurons go silent. 

FP-SAN v6 implements biological pupil dilation. The system continuously tracks an Exponential Moving Average (EMA) of `ambient_energy`. If the incoming energy drops, the network calculates a non-linear `homeostasis_ratio`:

$ratio = (current\_energy / expected\_energy)^{1.5}$

The global `spike_threshold` drops exponentially, hyper-sensitizing the neurons to recognize degraded signals without hallucinating new clusters.

## 3. Dynamic Z-Axis Neurogenesis (Solving Dimensionality)
Static architectures require pre-defined depth (e.g., ResNet-50). If the depth is wrong, the model either fails to abstract or wastes gigabytes of RAM. FP-SAN utilizes **Constructive Neural Architecture**. 

* **Layer V1** evaluates raw spatial data via localized receptive fields. 
* If V1 detects multiple distinct features simultaneously (e.g., a horizontal line AND a vertical line), the OS registers a "Composite Overload." 
* It dynamically spawns a **V2 Cortical Layer** in the Z-Axis. V2 does not read raw pixels; it exclusively reads the temporal spikes generated by V1, mathematically abstracting raw data into higher-order concepts in real-time.

## 4. Dopaminergic Action Selection (Basal Ganglia)
FP-SAN bypasses backpropagation for reinforcement learning by utilizing **Biological Eligibility Traces**. 

When a Sensory Cortex node fires and a random Motor Cortex node is triggered, the connecting synapse is flagged as active. If the environment returns a positive reward, a global Dopamine scalar (`+1.0`) floods the network, instantly crystallizing the active synapse. Penalties (`-1.0`) degrade the synapse. The network learns physical agency and survival instinct through localized chemical routing rather than global loss gradients.

## 5. Temporal Sequence Prediction (Neocortex)
To understand time, FP-SAN neurons contain **Lateral Dendritic Synapses**. When Cluster A spikes at $t=0$, it instantly forward-propagates a depolarizing current down its lateral synapse to Cluster B. 

At $t=1$, Cluster B is already "pre-charged" (predictive potential). This allows the architecture to predict physics and motion natively without heavy recurrent matrices like LSTMs or Transformers.

## 6. Orthogonal Neurogenesis (Catastrophic Forgetting)
When forced to learn new, contradictory domains, standard AI catastrophically overwrites historical weights. FP-SAN uses localized Spike-Timing-Dependent Plasticity (STDP). 

Only the winning cluster updates its weights. If incoming data completely mismatches all existing clusters (Orthogonal Rejection), the voltage drops below the Vigilance Threshold. Instead of forcing an update, the OS spawns a virgin cluster. Historical memory is permanently preserved.
